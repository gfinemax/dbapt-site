# 문서 자료실 즐겨찾기 및 관리 목록 개선 계획

## 목표

운영자의 전역 `중요` 표시와 사용자별 `즐겨찾기`를 명확히 분리하고, 조합원 개인자료실의 문서·게시글 저장 데이터가 모든 진입 화면에서 동일하게 보이도록 한다.

## 구현 범위

1. 홈·조합원·환불 포털에 `contentBookmarks` 전달을 연결한다.
2. 관리자 문서 로딩 시 현재 운영자의 문서 즐겨찾기 상태를 결합한다.
3. 전체 등록 문서 목록에 `내 즐겨찾기`, 공개 상태 필터, 통합 검색, 20건 페이지 구분을 추가한다.
4. `★` 헤더를 `중요`로 바꾸고 별도의 `즐겨찾기` 열과 모바일 액션을 제공한다.
5. 문서 개인 저장 용어를 `즐겨찾기` / `내 즐겨찾기`로 통일한다.
6. 관리자 진입 명칭은 `운영 문서 관리실`, 일반 사용자는 이름 기반 `개인 자료실`로 구분한다.
7. 관련 컴포넌트·API 연결 테스트를 갱신하고 lint, test, build 및 UI 검토를 수행한다.

## 제외 범위

- 문서 공개 승인 정책 변경
- DB 스키마 변경
- 감사 로그 삭제 또는 변경
- 공개 페이지의 접근 권한 확대
