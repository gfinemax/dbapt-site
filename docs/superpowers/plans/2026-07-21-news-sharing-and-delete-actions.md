# 소통마당 공유 링크 및 삭제 동작 구현 계획

## 목표

공지사항과 공개 공유가 허용된 자유게시판 글에 카카오톡 공유·링크 복사를 제공하고, 자유게시판 작성자와 관리자의 삭제 권한을 화면과 API에서 일치시킨다.

## 범위

- 기존 `/share/notice/[id]`, `/share/free/[id]` 미리보기 경로를 재사용한다.
- 상세 화면에 시스템 공유창 기반 `카카오톡 공유`와 `링크 복사`를 제공한다.
- 자유게시판은 공개 공유가 허용된 실제 게시글만 외부 공유한다.
- 자유게시판 작성자와 관리자는 목록 및 상세에서 삭제할 수 있다.
- 공지사항 삭제는 기존 정책대로 관리자만 가능하며 상세 화면에서 찾을 수 있게 한다.
- 권한 검사는 기존 서버 API에서 다시 수행한다.

## 제외

- 카카오 JavaScript SDK와 앱 키 신규 도입
- 카카오톡 자동 게시 또는 메시지 자동 발송
- 비공개 자유게시판 게시글의 외부 공개
- 삭제 복구함 또는 소프트 삭제 도입

## 검증

- 공유 URL, 시스템 공유, 클립보드 복사 단위 테스트
- 작성자·관리자·다른 사용자별 삭제 버튼 노출 테스트
- 삭제 API 회귀 테스트
- `pnpm lint`, `pnpm test`, `pnpm build`
- 데스크톱·모바일 화면과 키보드 포커스 확인
