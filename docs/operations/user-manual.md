# 대방동지역주택조합 사이트 사용자 매뉴얼

작성일: 2026-06-25

## 개인자료실 보관

로그인한 사용자는 공지사항, 조합뉴스, 자유게시판 글과 공개자료 문서를 개인자료실에 보관할 수 있다.

### 게시글 보관

1. `조합소식`에서 공지사항, 조합뉴스, 자유게시판 글 목록을 연다.
2. 보관하려는 글의 `보관` 버튼을 누른다.
3. 저장된 글은 개인자료실의 `보관한 게시글` 탭에서 다시 볼 수 있다.
4. 다시 누르면 보관이 해제된다.

게시글 보관은 로그인한 본인 계정에만 적용된다. 다른 사용자의 보관 목록이나 게시글 공개 상태는 바뀌지 않는다.

### 공개자료 문서 보관

1. `정보공개` 또는 공개자료 메뉴에서 `등록 자료` 영역이나 `자료실 열기` 목록을 확인한다.
2. 보관하려는 문서의 `보관` 버튼을 누른다.
3. 저장된 문서는 개인자료실의 `내 보관함` 탭에서 확인한다.
4. `보관 해제`를 누르면 개인자료실에서 제거된다.

문서 보관은 문서 승인 상태, 공개 범위, 관리자 메타데이터를 변경하지 않는다.

## PDF 열람

문서의 `열람`을 누르면 PDF 뷰어가 열린다.

- `PDF만 크게` 버튼: 제목, 설명, 부가 정보 영역을 접고 PDF 화면만 크게 본다.
- `상세 보기` 또는 `화면 축소` 버튼: 원래 상세 정보가 포함된 보기로 돌아간다.
- `다운로드` 버튼: 권한이 있는 문서를 내려받는다.
- `닫기` 버튼: PDF 뷰어를 닫는다.

개인자료실 오른쪽 서랍에서 문서를 열어도 PDF 뷰어는 서랍 안이 아니라 전체 화면 레이어로 열린다.

## 관리자 PDF 업로드 저장 방식

관리자가 PDF 문서를 업로드하거나 공개자료 첨부를 교체할 때 기본값은 `자동 최적화`다.

### 자동 최적화 기준

- PDF가 5MB 이하이면 원본 그대로 저장한다.
- PDF가 5MB를 초과하면 안전한 구조 최적화를 시도한다.
- 최적화 결과가 원본보다 15% 이상 작을 때만 최적화본을 저장한다.
- 기준에 못 미치면 원본을 저장한다.
- 원본과 최적화본을 동시에 저장하지 않고 최종 저장 파일 1개만 유지한다.

### 원본 그대로 저장이 필요한 경우

도면, 스캔본, 직인, 서명, 이미지 품질이 중요한 문서는 업로드 화면에서 `원본 그대로 저장`을 선택한다.

현재 최적화는 PDF 내부 구조를 정리하는 안전한 방식이다. 스캔 이미지를 강제로 낮은 해상도로 바꾸거나 PDF를 이미지로 변환하지 않는다.
